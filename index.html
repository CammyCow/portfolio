ChatApp

Java application that allows users to remotely create chatrooms and message other users. The first connection to the "Discovery Server" allows an individual's ip to be easily discoverable by other users. The second connection is to individual people, in which connection stubs are exchanged to allow messaging. Features "auto-connect-back", which allows a newly connected user to receive all the connection stubs of a user with pre-existing user connections. The ChatApp API also allows the creation of custom messages that can take advantage of Java Swing elements, such as button creation. See demo video above.
